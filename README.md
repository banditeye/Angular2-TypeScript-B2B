# Angular 2 TypeSript for B2B project

Serve frontend app by webpack-dev-server.

```
npm install -g yarn
yarn install
yarn start
open http://localhost:4200
```

Testing.

```
yarn test
```

Production build.

```
yarn run build:prod
yarn run server:prod
open http://localhost:4200
```
